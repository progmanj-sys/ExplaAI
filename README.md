How to install ExplaAI: Step-by-step instructions
Installation usually takes less than a minute. Follow these simple steps:
1. Go to the extension store
Open Google Chrome and go to the Chrome Web Store. You can simply enter the name of the extension in the search bar or follow a direct link if you have one.
2. Search for the extension
In the search field (top left), type “ExplaAI” and press Enter. Select the appropriate extension from the list of results.
3. Add to browser
On the extension page, click the large blue “Add to Chrome” button.
4. Confirm installation
A pop-up window will appear asking for confirmation. Click “Add extension.” The browser will download and automatically install the file.
5. Pin for quick access
After installation, the extension icon may be “hidden.” To keep it handy:
Click on the puzzle icon (Extensions) in the upper right corner of the browser.
Find ExplaAI in the list.
Click on the pin icon to pin it to the toolbar.
